# Juego_SimonDice_TresRaya
Este repositorio contiene dos juegos clásicos desarrollados con HTML y CSS: Simon Dice y Tres en Raya. El proyecto está enfocado en practicar estructuras HTML, estilos CSS y lógica básica del juego del lado del cliente, sin uso de frameworks externos. Ideal como proyecto de aprendizaje y demostración de fundamentos de desarrollo web.
